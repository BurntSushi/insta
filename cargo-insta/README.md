`cargo-insta` provides a cargo command for [insta](https://github.com/mitsuhiko/insta)
snapshot reviews.

```
$ cargo install cargo-insta
$ cargo insta --help
```

For more information see [the insta crate documentation](https://docs.rs/insta).
